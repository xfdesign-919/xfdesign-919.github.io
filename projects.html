Projects

Onestep

Swift, SwiftUI, Apple Family Controls API

Onestep is an iOS app I designed and built solo to interrupt compulsive screen time through small moments of friction rather than blunt-force blocking. Instead of simply locking an app, Onestep asks the user to clear a short challenge first, giving them a real moment to decide whether they actually want to open it. Building it meant working through Apple's Family Controls and DeviceActivity frameworks, resolving persistent SwiftData state bugs, and iterating through several rounds of beta testing to land on an approach that actually changed behavior instead of just adding friction for its own sake.

M.B.A.S. — Morning Briefing Automation System

Python, Bash, JavaScript, OAuth 2.0

A fully automated macOS system that builds and delivers a personalized morning briefing using launchd scheduling, bash orchestration, and Python. It pulls from the Google Calendar API and the Open-Meteo weather API into a custom HTML/CSS/JS dashboard. Most of the actual engineering work here was less about writing new code and more about directing and pressure-testing AI-generated code, catching subtle bugs, and rebuilding pieces that looked right but didn't actually work.

MountSQL Hospital Database System

MySQL, ER Modeling

A normalized relational database designed to modernize hospital recordkeeping across departments, physicians, patients, and procedures. The project involved modeling complex many-to-many relationships, such as physician training assignments and nurse scheduling, using junction tables, and resulted in a full schema of 10+ interrelated tables supporting efficient querying of medical records and appointment history.
